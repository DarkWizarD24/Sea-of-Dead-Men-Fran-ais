## Modifications

Vous pouvez modifier cette traduction et généré les PDF.

Vous devez d'abord installer les logiciels suivants :
  * gimp
  * scribus
  * python 3
  * PyPDF2 (pip install PyPDF2)

Lancez la génération avec la commande `py export.py`
